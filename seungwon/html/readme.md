# HTML 학습하기 

## 문서의 구성

html 문서는 DTD를 선언하고, html 태그를 열고 그 안에 head 와 body로 구분하여 작성한다.

<!-- ```- > backTab -->
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```
